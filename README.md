# Case Técnico – Grupo Moura

Este repositório contém a resolução do case técnico proposto pelo Grupo Moura, utilizando SQL (MySQL) para análise dos dados e Power BI para construção do dashboard.

## Estrutura do projeto

- `Queries-Desafio` → Queries desenvolvidas para responder às questões do case;
- `Analise_Case.pdf` → Documento contendo as queries, resultados e análises realizadas;
- `Dashboard.png` → Arquivo PNG do Dashboard;
- `Dashboard-Case-Tecnico.pbix` → Arquivo do dashboard desenvolvido no Power BI;
- `desafio_moura.sql` → Arquivo SQL com a base de dados utilizada em MySQL;
- `bases/` → Arquivos CSV disponibilizados para o desafio.

## Atividades desenvolvidas

### 1. Produtos vendidos abaixo do preço de tabela
Identificação de produtos comercializados com preço inferior ao cadastrado, buscando entender possíveis impactos comerciais.

### 2. Análise de desempenho do colaborador Robert King em 2022
Levantamento da quantidade de pedidos realizados e do faturamento gerado pelo colaborador durante o período.

### 3. Top 10 produtos mais caros
Listagem dos produtos com maior preço unitário para análise do portfólio.

### 4. Comparação do faturamento dos fornecedores
Análise da evolução do faturamento dos fornecedores entre os anos de 2020 e 2021, destacando aqueles que apresentaram maior crescimento.

### 5. Dashboard em Power BI
Desenvolvimento de um painel interativo contendo:

- Faturamento Total;
- Quantidade de Pedidos;
- Ticket Médio;
- Itens Vendidos;
- Top 5 Categorias com Maior Receita;
- Filtro por ano.

## Considerações

O desenvolvimento deste case permitiu aplicar conhecimentos em SQL, análise de dados e Power BI para responder perguntas de negócio e construir indicadores que auxiliam na tomada de decisão.
